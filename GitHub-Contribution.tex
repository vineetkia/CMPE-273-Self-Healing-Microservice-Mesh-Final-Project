\subhead{Testing contribution (shared)}
The \texttt{tests/} directory contains 69 pytest unit tests covering:
\begin{itemize}
  \item Resilience primitives (retry-with-backoff, circuit breaker state machine)
  \item Failure-mode chaos primitive (errors / latency / grey modes, auto-expiry)
  \item Healer 2-of-3 consensus rule and threshold boundaries
  \item Root-cause graph walk for distinguishing symptom from cause
  \item Action allowlist invariant (4 verbs $\times$ 8 services = 32 actions)
  \item Telemetry sliding window, error rate, p95 latency with sub-ms precision
  \item Auth password hashing (salt + SHA256) and user-ID derivation
\end{itemize}
Run via \texttt{make test} or \texttt{python3 -m pytest tests/}. Total runtime: under 2 seconds.
